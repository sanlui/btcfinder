# BTC Finder

A technical Bitcoin blockchain explorer tool.

## Features
- Search transactions by hash
- Lookup addresses
- View block details
- Clean, technical interface

## Technologies
- HTML5, CSS3, JavaScript
- Blockstream API
- Mempool.space API

## Setup
1. Clone repository
2. Open `index.html` in browser
3. Or deploy to GitHub Pages/Vercel/Netlify
